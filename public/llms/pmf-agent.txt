# PMF Agent · Product-Market Fit Score

> Есть ли у твоей идеи product-market fit? 9-осевой скоринг с честным вердиктом GO / VALIDATE / PIVOT.

URL: https://founders-circle.space/agents/pmf
API: https://pmf-api.founders-circle.space

## Что делает

PMF Agent - pipeline из 12 AI-агентов, который оценивает идею по 9 осям и выдаёт PMF-Score, вердикт и (если score низкий) альтернативные траектории через Pivot Advisor.

## 9 осей PMF Score

1. Pain - насколько острая боль у целевого сегмента
2. Market - размер и доступность рынка
3. Timing - почему именно сейчас
4. Competitive whitespace - есть ли свободная позиция
5. Demand - живой спрос, а не выдуманный
6. Monetization - готовность платить и модель
7. Unit economics - сходится ли экономика
8. Regulatory - правовые/регуляторные риски
9. Execution - выполнимо ли командой/одним фаундером

## Вердикты

- GO - score высокий, идея жизнеспособна, дальше - валидация на клиентах
- VALIDATE - сомнительные оси, нужно проверить через интервью/MVP
- PIVOT - score < 50, Pivot Advisor предлагает 2-3 альтернативные траектории на смежные сегменты или модели

Агент честный, не льстит. Если идея слабая - скажет, почему.

## Параметры запуска

- LLM-провайдер: Anthropic / OpenAI / Google
- Модель: выбор из каталога + кастомный ID
- API-key: используется только в сессии, не сохраняется
- Язык отчёта: русский или английский
- Идея (1-3 предложения) + опциональный контекст: ЦА, география, монетизация, референсы

## Время и стоимость

- Время прогона: ~10-15 минут
- Стоимость: с твоего LLM-ключа (порядок $1-3 на Claude 4.x, дешевле на Haiku/Gemini Flash)

## Как работает

State machine: setup -> clarify (если intake видит нехватку контекста) -> progress (SSE-стрим событий агентов) -> report (PMF-Score + вердикт + Pivot-сценарии при низком score)
